مشروع جامعة - ملفات جاهزة

الملفات داخل المجلد:
- db.sql            : استيراد في phpMyAdmin لإنشاء قاعدة البيانات والجداول + بيانات تجريبية
- config.php        : إعدادات اتصال بقاعدة البيانات (ضع اسم المستخدم/كلمة السر لو لازم)
- index.php         : صفحة تسجيل الدخول
- register.php      : صفحة إنشاء حساب
- courses.php       : صفحة تسجيل المواد (بعد تسجيل الدخول)
- logout.php
- assets/css/custom.css
- ERD.png           : صورة مبسطة لمخطط ERD
- Data_Dictionary.xlsx : ملف Excel يوضح أعمدة الجداول وأنواعها ووصفها

كيفية التشغيل محلياً:
1) ضع المجلد في مجلد htdocs لو تستخدم XAMPP أو www لو تستخدم WAMP.
2) افتح phpMyAdmin واستورد الملف db.sql (Import) أو أنشئ قاعدة uni_app ثم استورد.
3) افتح المتصفح واذهب إلى: http://localhost/uni_project/index.php
4) يمكنك إنشاء حساب جديد ثم تجربة تسجيل المواد.

ملاحظة: إذا كان اسم مجلد المشروع مختلفًا، غيّر المسار في المتصفح ليناسب.

بالتوفيق — لو عايز أظبط أي حاجة أو أعدل نصوص عربية أو أضيف صفحة admin قولّي.
